This project introduces more advanced Python data structures, focusing on sets and dictionaries.
learning how to store collections of data, access them efficiently, and manipulate them through built-in Python features.
tasks  in this project:

File	Description
0-square_matrix_simple.py	Computes the square value of all integers in a matrix
1-search_replace.py	Replaces all occurrences of an element with another
2-uniq_add.py	Adds all unique integers in a list
3-common_elements.py	Returns a set of common elements between two sets
4-only_diff_elements.py	Returns elements present in only one set
5-number_keys.py	Returns the number of keys in a dictionary
6-print_sorted_dictionary.py	Prints a dictionary ordered by keys
7-update_dictionary.py	Replaces or adds a key/value in a dictionary
8-simple_delete.py	Deletes a key from a dictionary
9-multiply_by_2.py	Returns a dictionary with all values multiplied by 2
10-best_score.py	Returns the key with the highest value in a dictionary
11-multiply_list_map.py	Uses map() to multiply list values
12-roman_to_int.py	Converts a Roman numeral to an integer
