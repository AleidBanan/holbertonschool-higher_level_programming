# MySQL - Basic SQL Scripts

This project contains simple SQL scripts to practice basic MySQL commands such as creating databases, creating tables, selecting data, and using SQL functions like `COUNT()` and `AVG()`.

Each task is written in its own `.sql` file and can be executed using the MySQL CLI.

---

## Requirements

- Ubuntu 20.04 / 22.04
- MySQL 8.0+
- All SQL files:
  - end with a new line
  - contain valid SQL syntax
  - use `--` for comments when needed

---

## How to Run

### Start MySQL (sandbox)
```bash
service mysql start
