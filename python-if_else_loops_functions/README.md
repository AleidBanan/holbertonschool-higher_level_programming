| Task | Description |

| 1 | The last digit |
| 2 | Alphabet game for insomnia (print letters A-Z) |
| 3 | Alphabet soup payoff (string manipulation) |
| 4 | Hexadecimal printing |
| 5 | 00...99 (printing numbers with leading zeros) |
| 6 | Inventing is a combination of brains and materials |

| 11 | Bitwise XOR of two numbers (`a ^ b`) |
